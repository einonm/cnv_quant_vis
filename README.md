# Visualiser for the effect of CNVs on Quantitative Traits

**Required packages**
- bokeh
- scipy
- pandas
- numpy
- statsmodels
- pyarrow


**Or simply:**
- git clone https://github.com/seafloor/cnv_quant_vis.git
- cd cnv_quant_vis
- conda env create -f environment.yml

Data must be in cnv_quant_vis/data/biobank2.parquet
